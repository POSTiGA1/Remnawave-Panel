## Хэдеры

Хэдеры – это заголовки, которые отправляются вместе с подпиской (ответом).

В Remnawave вы можете использовать ключ `rwEncodeBase64:` для того, чтобы закодировать значение хедера в base64.

Ниже приведен список популярных хедеров. Некоторые из них настроены по умолчанию.

### profile-title

Хэдер, который в основном используется для отображения названия профиля в клиентском приложении.

Ключ: `profile-title`

Пример значения: `rwEncodeBase64:Моя крутая подписка`

Значение по умолчанию: `rwEncodeBase64:Remnawave`

### profile-web-page-url

Используется для отображения ссылки на веб-страницу профиля в клиентском приложении.

Ключ: `profile-web-page-url`

Пример значения: `rwEncodeBase64:https://dummy.docs.rw`

Значение по умолчанию: `rwEncodeBase64:{{SUBSCRIPTION_URL}}`

### profile-update-interval

Используется для установки интервала обновления подписки в клиентском приложении.

Ключ: `profile-update-interval`

Пример значения: `1`

Значение по умолчанию: `12`

### support-url

Используется для установки ссылки на поддержку в клиентском приложении.

Ключ: `support-url`

Пример значения: `https://dummy.docs.rw`

Значение по умолчанию: `https://dummy.docs.rw`
